# Frontend

The frontend is a minimal React interface for uploading images and viewing results.

## Features

- File upload input.
- Image preview.
- Loading state during upload.
- Similarity percentage.
- Duplicate status.
- Owner display.
- Blockchain verification indicator.
- Red warning when duplicate content is detected.

## Important Files

- `src/App.jsx`
- `src/components/Uploader.jsx`
- `src/components/ResultCard.jsx`
- `src/components/StatusPill.jsx`
- `vite.config.js`

## Local Run

```bash
npm install
npm run dev
```
